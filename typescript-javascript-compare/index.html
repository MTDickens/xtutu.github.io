前言

TypeScript官方文档：http://www.typescriptlang.org/Handbook
首先需要明确的是TypeScript并不是一个真正的语言，准确的说它只是JavaScript的一个语法糖(syntactic sugar)。用TypeScript写的代码，最终还是要转化为JavaScript来执行的。
所以学习TypeScript之前， 一定要学JavaScript。
